# Advanced Multipeer Messaging Playground (Swift)

This repo contains a ready-to-upload Swift Playground that you can publish to GitHub and share as a downloadable app sample.

## Files
- `MessagingPlayground.playground/` → open this directly in Swift Playgrounds or Xcode.

## GitHub sharing quick steps
1. Push this repo to GitHub.
2. Users can click **Code → Download ZIP** to get everything.
3. (Optional) Add a GitHub Release and attach your own zip export.

## Create a release zip locally
Run this from repo root:

```bash
zip -r MessagingPlayground.zip MessagingPlayground.playground
```

## App highlights
- Multipeer peer discovery and secure messaging
- iMessage-inspired chat bubble UI
- Typing indicators
- Read receipts (sent/delivered/read)
- Emoji reactions
